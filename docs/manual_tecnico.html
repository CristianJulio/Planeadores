Descripción de la plataforma

Planeadores es una Web que fue creada con la base de Ruby, CSS, Bootstrap, Javascript y Jquery. Para tener un buen rendimiento de la plataforma se requiere de lo siguiente:

  1. Sistema operativo Windows 7 o posteriores, Linux Kernel 3.4 en adelante o Mac OS v10 o posterior.
  2. Computador con mínimo de 4 GB en RAM.
  3. Espacio libre en disco de 500 MB.
  4. Impresora láser (para impresión de listados).
  5. Navegador Chrome versión 30 en adelante, Firefox versión 20 en adelante.
  6. Lector de archivos PDF como Acrobat Reader o similar.
